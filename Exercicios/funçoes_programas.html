HTML: O HTML auxilia no desenvolvimento web, organizando as páginas e dando personalidade às mesmas. Assim, o usuário consegue ter acesso a imagens, textos e diversos elementos que, sem o HTML, não seriam carregados. Sendo utilizado no desenvolvimento front-end das aplicações web, tanto para Android quanto para iOS. CSS: O CSS é usado para estilizar elementos escritos em uma linguagem de marcação como HTML, como separar o conteúdo do site de sua apresentação visual, alterando elementos como cor do texto, fonte e espaçamento entre blocos, assim como todo o aspecto estético de uma página. Utilizado em conjunto com o HTML para a construção de páginas. JavaScript: O JavaCript é uma linguagem de programação que nos permite executar instruções como resposta às ações do usuário como, por exemplo, atualizar o conteúdo de uma página web sem precisar recarregá-la por completo ao preencher um formulário. Pondendo ser usado no Front-end, Back-end e até mesmo na comunicação com banco de dados.
